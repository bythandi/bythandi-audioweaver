# bythandi-audioweaver
Convert PDFs and text to accessible audio — free and unlimited.
Features
• PDF Upload — Extract text from PDF documents
• Text Input — Paste or type text directly
• 5 Languages — English, Portuguese, Japanese, French, Spanish • Speed Control — Normal or Slow playback
• Format Options — MP3 or WAV output
• Progress Tracking — Visual feedback during conversion
Tech Stack
• Streamlit — Web interface
• gTTS — Google Text-to-Speech (free, no API key) • pdfplumber — PDF text extraction
• pydub — Audio processing
Local Development
   pip install -r requirements.txt
   streamlit run app.py
Deployment
Deploy free on Streamlit Cloud (streamlit.io/cloud):
1. Push this repo to GitHub
2. Connect to Streamlit Cloud
